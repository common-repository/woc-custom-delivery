== Description ==

This plugin allows your customer to make order selecting a delivery time on that day.

== Installation ==

1. Upload Woocommerce Custom Delivery to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Now select from the Admin menu Woocommerce Custom Delivery and edit your settings.
